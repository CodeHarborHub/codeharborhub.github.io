1. **Font Sizes**:
   - Tailwind CSS offers a variety of font sizes that you can apply using the `text-{size}` class.
   ```html
   <p class="text-xs">Extra small text</p>
   <p class="text-sm">Small text</p>
   <p class="text-base">Base text (default)</p>
   <p class="text-lg">Large text</p>
   <p class="text-xl">Extra large text</p>
   <p class="text-2xl">2x large text</p>
   <p class="text-3xl">3x large text</p>
   <p class="text-4xl">4x large text</p>
   <p class="text-5xl">5x large text</p>
   <p class="text-6xl">6x large text</p>
   ```

2. **Font Weights**:
   - Tailwind CSS provides classes for different font weights using the `font-{weight}` class.
   ```html
   <p class="font-thin">Thin weight</p>
   <p class="font-extralight">Extra light weight</p>
   <p class="font-light">Light weight</p>
   <p class="font-normal">Normal weight (default)</p>
   <p class="font-medium">Medium weight</p>
   <p class="font-semibold">Semi-bold weight</p>
   <p class="font-bold">Bold weight</p>
   <p class="font-extrabold">Extra bold weight</p>
   <p class="font-black">Black weight</p>
   ```

3. **Font Styles**:
   - You can italicize text using the `italic` class and remove italics using the `not-italic` class.
   ```html
   <p class="italic">Italic text</p>
   <p class="not-italic">Not italic text</p>
   ```
